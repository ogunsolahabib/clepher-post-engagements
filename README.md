# Clepher's Posts engagement page (Frontend)

## About
Replica of the posts engagement page

## Technologies
- React
- TypeScript
- React Router
- Tailwind (Daisy U)

Page hosted at [https://clepher-post-engagements.vercel.app/](https://clepher-post-engagements.vercel.app/)



## How to run locally
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
